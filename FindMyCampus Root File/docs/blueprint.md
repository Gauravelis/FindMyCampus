# **App Name**: Campus Compass

## Core Features:

- College Directory: Browse a searchable and filterable directory of colleges.
- College Details: View detailed information about each college.
- Admin Authentication: Secure admin login using Firebase Authentication.
- College Management: Admin dashboard to Create, Read, Update, and Delete college listings.
- Data Import: Admin tool to bulk-import colleges from a CSV file. A reasoning AI may be employed as a tool to perform rudimentary sanitization on imported fields before they are saved.
- Image Upload: Secure authenticated image uploads for college hero images and galleries.
- Search & Filter: Filter and search colleges by name, city, state, courses, and facilities.

## Style Guidelines:

- Primary color: Muted blue (#6699CC) evokes trust and knowledge, fitting for an educational app.
- Background color: Very light blue (#F0F8FF) creates a clean, professional backdrop.
- Accent color: Soft lavender (#B39DDB) complements the primary blue, offering visual interest without being overwhelming.
- Body text: 'PT Sans', a humanist sans-serif font suitable for continuous reading.
- Headline Font: 'Space Grotesk', a sans-serif font for headers to convey a contemporary and techy feel. If the design requires blocks of text that are more than one paragraph in length, please use 'PT Sans' for body text instead.
- Code font: 'Source Code Pro' for displaying code snippets (for documentation purposes).
- Use clear, modern icons (e.g., from Lucide) to represent facilities and amenities.
- Mobile-first, responsive layout optimized for various screen sizes.
- Subtle animations for loading states and transitions to enhance user experience.
